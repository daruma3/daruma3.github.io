問題概要

床がR*Cマスに分割されています．上下左右の移動により全てのマスをK回ずつ踏むことができる場合は"Print"を， そうでない場合は"Cannot print"を返しなさい．

考え方

K==1の時は自明．そうでない場合について考える．実際にやってみると分かるが，ただくるくる回るだけだとスタート地点とゴール地点で 回数が足りなくなる．そのため全ての行または列についてK往復することを考えるがそれでも端のマスだけ足りない．端のマスを一度に 二度踏めないからだ．そこで逆に端しかない場合を考える．つまり二マスセットを作り両方K回踏んだら次の二マスに移る．

うん，これならうまくK回ずつ踏めそうだ．だから踏める条件は，K>2の時は二マスセットを作れるR*C%2==0の時となる．

コード

  1. #include <bits/stdc++.h>
  2. using namespace std;
  3. class PaintTheRoom {
  4.     public:
  5.     string canPaintEvenly(int R, int C, int K) {
  6.         if(R*C%2==0 or K==1) return "Paint";
  7.         else return "Cannot paint";    }
  8. };
