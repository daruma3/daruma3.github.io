問題概要

問題

夏休みN日間をA～C，三つのタスクのいづれかを毎日行うことで実りあるものにすることにしました． i日目におけるこれらのタスクの幸福度はai,bi,ciで定められており， また飽きっぽいので二日連続同じ種類のタスクを行うことはできない．幸福度の最大値を求めよ．

反省

前日のを使えないということから二日前も関係あるのか?と思い最初三項間漸化式かと思った()．普通に 漸化式っぽいのを思いつくまではいったものの前日行ったタスクの保持方法が思いつかず撃沈．

考察

二日連続がだめということは，前日に異なるタスクを選ばなければいけない．漸化式は
dp[i+1][j]=max(dp[i+1][j],dp[i][k]+happy[i][j])(ただしj!=k)
となる．これをループさせていけばよい．

コード

  1. #include<bits/stdc++.h>
  2. using namespace std;
  3. const long long INF=1000000000;
  4. typedef long long ll;
  5. #define rep(i, n) for(int i = 0; i < n;i++)
  6. ll n,dp[100005][3];
  7. int main(){
  8.    cin>>n;
  9.    ll happy[n][3];
  10.    rep(i,n) cin >> happy[i][0] >> happy[i][1] >> happy[i][2];
  11.     rep(i,n)
  12.         rep(j,3) dp[i][j] = -INF;
  13.     rep(j,3) dp[0][j] = 0;
  14.     rep(i, n){
  15.         rep(j, 3){
  16.             rep(k, 3){
  17.                 if (j == k) continue;
  18.                 dp[i + 1][j] = max(dp[i + 1][j], dp[i][k] + happy[i][j]);
  19.             }
  20.         }
  21.     }
  22.     ll ans = -INF;
  23.     rep(i, 3) ans = max(ans, dp[n][i]);
  24.     cout << ans << endl;
  25.     return 0;
  26. }
